# 🚀 NEXUS Quantitative Engine

NEXUS is an institutional-grade, multi-threaded MetaTrader 5 trading orchestrator written in Python. It dynamically routes real-time market data (tick processing) into 5 distinct quantitative and AI-filtered engines based on the current mathematical market regime (Trend, Range, Dead Volatility, Breakout).

## 🧠 System Architecture

The core philosophy of the NEXUS engine is modular strategy routing. The Orchestrator automatically tracks the `ADX` (Average Directional Index) and Time of Day logic (e.g. London / New York Overlaps). 

If the market chops sideways, the Trend algorithms are forcibly sheathed, and Mean Reversion engines are deployed. If momentum breaches historical levels, Breakout engines are unholstered.

### The 5 Quant Engines
1. **SMC Order Block (`strategies/smc_orderblock.py`)**: Structurally detects BOS (Break of Structure) and draws protective institutional entries on the final opposing candle.
2. **SMC Fair Value Gaps (`strategies/smc_fvg.py`)**: Detects volumetric inefficiencies and executes only alongside macro timeframe confluence.
3. **Bollinger Volatility Squeeze (`strategies/bb_breakout.py`)**: Waits for historical kinetic compression in the Bolinger Bandwidth (lowest 15% out of 100 periods) to blindly jump onto exploding momentum vectors.
4. **VWAP Reversion (`strategies/vwap_reversion.py`)**: Tracks High-Frequency (HFT) standard deviations off the VWAP line, mean-reverting any violent > 3.0 Standard Deviation rubber-band snaps.
5. **RSI AI Scalping (`strategies/rsi_reversion.py`)**: Executes strict limit limits on local chop zones using proprietary Random Forest Classifiers.

## 🔐 The AI Gatekeeper

NEXUS differs from traditional programmatic bots by validating its algorithmic setups against pre-trained Machine Learning memory banks. 
Every trade signal generated by the standard algorithms MUST pass a `.predict_proba()` query on the Scikit-Learn `.joblib` Random Forest model. If the artificial intelligence predicts historical failure and outputs a confidence score `< 0.55`, the execution is instantly rejected by the macro `AI DENIED` trigger.

## 🛠️ Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Ugochi56/Nexus-Trading-Bot.git
   ```

2. **Install Computational Dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Provide Intelligence Cache:**
   *Note: Because `.joblib` AI training files are excluded via `.gitignore` for security/privacy, ensure you place the `gold_trend_model.joblib` and `gold_reversal_model.joblib` directly into the root directory before running the bot!*

## ⚙️ Configuration
Open `core/config.py` to globally dictate the bot's risk paradigm.
```python
SESSION_RISK = {
    "ASIAN": 0.5,             # 0.5% Risk
    "LONDON": 1.0,            # 1.0% Risk
    "NY_LONDON_OVERLAP": 0.0, # FLAT (Algorithm refuses to trade the dangerous chop)
    "NEW_YORK": 1.0           # 1.0% Risk
}
AUTO_DST_ADJUST = True        # Shifts session parameters during Daylight Savings
```
